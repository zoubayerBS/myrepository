# VacationEasy

VacationEasy est une application web complète conçue pour simplifier la gestion des vacations, particulièrement adaptée au milieu médical. Elle permet aux utilisateurs de soumettre leurs vacations et aux administrateurs de les gérer, tout en offrant des outils de communication et de reporting.

## Fonctionnalités Principales

### Pour tous les utilisateurs :
- **Authentification Sécurisée :** Inscription et connexion avec gestion de session.
- **Messagerie Intégrée :** Un système de chat en temps réel pour faciliter la communication entre les utilisateurs.

### Pour les Utilisateurs (Employés) :
- **Tableau de Bord Personnel :** Une vue d'ensemble de leurs vacations du mois en cours et de celles en attente de validation.
- **Soumission de Vacations :** Un formulaire simple pour enregistrer de nouvelles vacations (actes ou forfaits).
- **Suivi de Statut :** Visualisation en temps réel du statut de leurs demandes (En attente, Validée, Refusée).
- **Historique :** Accès à l'historique des vacations passées (validées ou refusées).
- **Rapports Personnels :** Génération de rapports de vacations sur une période donnée.

### Pour les Administrateurs :
- **Tableau de Bord Complet :** Vue globale sur toutes les vacations, statistiques clés et classement des employés.
- **Gestion des Vacations :** Validation, refus et gestion de toutes les demandes de vacation des utilisateurs.
- **Gestion des Utilisateurs :** Ajout, modification ou suppression des utilisateurs de la plateforme.
- **Archivage :** Fonctionnalité pour archiver les anciennes vacations et consulter les archives.

## Technologies Utilisées

- **Framework Frontend :** Next.js (avec React et TypeScript)
- **Style :** Tailwind CSS & shadcn/ui
- **Backend :** API Routes de Next.js
- **Base de données :** Supabase
- **Tests :** Jest & Testing Library

Ce projet a été développé avec une architecture moderne et scalable pour assurer une expérience utilisateur fluide et une maintenance aisée.
